# O’Neill Tensor Curvature Codes

This repository contains computational and numerical codes associated with
the study of sectional curvature structures on fibred Calabi–Yau manifolds
using Riemannian submersions and O’Neill tensor decompositions.

## Scope
The codes support analytical and numerical investigations of:
- O’Neill A- and T-tensors
- Horizontal–vertical curvature decomposition
- Elliptic, toroidal, and K3 fibrations
- Curvature bounds and collapsing limits

## Associated Work
These codes accompany the preprint/monograph:
**“Submersion Bounds on Fibred Calabi–Yau Geometry via O’Neill Tensors.”**

## Repository Structure
- `src/` — core geometric routines
- `numerics/` — numerical curvature computations
- `ml/` — machine-learning–based methods
- `data/` — auxiliary datasets
- `examples/` — illustrative examples
